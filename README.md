# Система оценки профессиональных рисков

Автоматизированная система для генерации карт оценки профессиональных рисков и отчётов.

## Системные требования

- **Python 3.8 или новее**
- **Операционная система:** Windows 10/11, Linux, macOS
- **Свободное место на диске:** ~500 МБ
- **Оперативная память:** 2 ГБ (рекомендуется 4 ГБ)

## Быстрый старт

### Для Windows

1. **Скачайте проект:**
   - Нажмите зелёную кнопку `Code` → `Download ZIP`
   - Распакуйте архив в любую папку

2. **Установите Python** (если ещё не установлен):
   - Скачайте с https://www.python.org/downloads/
   - **ВАЖНО:** При установке отметьте галочку **"Add Python to PATH"**

3. **Запустите программу:**
   - Дважды кликните на файл `start.bat`
   - Подождите, пока установятся все необходимые библиотеки (только при первом запуске)

4. Откройте браузер и перейдите по адресу: `http://127.0.0.1`

## Инструкция по использованию

### Шаг 1: Загрузка файлов
1. На главной странице укажите дату документа
2. Загрузите необходимые файлы:
   - **Шаблон карты** (card_template.docx)
   - **Шаблон отчёта** (rep_template.docx)
   - **Список сотрудников** (people.xlsx)
   - **Данные организации** (org.xlsx)

### Шаг 2: Выбор опасностей
- Отметьте галочками опасности, которые присутствуют на рабочих местах
- Нажмите "Продолжить"

### Шаг 3: Оценка рисков для каждого работника
- Для каждого работника заполните:
  - **Степень риска** (deg)
  - **Вероятность** (ch)
  - **Коэффициент** (kef)
- Нажмите "Сохранить и далее"

### Шаг 4: Скачивание результатов
- После обработки всех работников нажмите **"Скачать ZIP-архив"**
- В архиве будут:
  - Карты оценки рисков для каждого работника
  - Итоговый отчёт

### Шаг 5: Завершение работы
- Нажмите кнопку **"Завершить работу"** для:
  - Удаления временных файлов
  - Остановки сервера
  - Закрытия приложения

## Решение проблем

### Python не найден
- **Windows:** Переустановите Python с галочкой "Add Python to PATH"
